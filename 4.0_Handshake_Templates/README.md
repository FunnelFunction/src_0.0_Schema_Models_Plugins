# 4.0 Handshake Templates

> **"Protocol-OS ready templates."**

Pre-configured templates for the FunnelFunction Librarian's Protocol-OS system.

## Folder Structure

```
4.0_Handshake_Templates/
├── curl_templates/     ← Universal cURL mode templates
└── oauth_templates/    ← OAuth flow configurations
```

## Usage

These templates plug directly into Librarian's Sync tab. Each template includes:
- Complete cURL command with `{INPUT}` placeholder
- Authentication configuration
- Response parsing instructions
